# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| 0.1.x   | :white_check_mark: |
| < 0.0.x   | :x:                |

## Reporting a Vulnerability

When reporting a vulnerability, please use the Issues system and tag the issue as a vulnerability.
